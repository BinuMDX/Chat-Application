# Chat Application

A real-time chat application built with modern technologies.

## Project Structure

- `chat-backend/` - NestJS backend server

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or pnpm

### Installation

```bash
# Install backend dependencies
cd chat-backend
npm install
```

### Running the Application

```bash
# Run backend in development mode
cd chat-backend
npm run start:dev
```

## Technologies Used

- **Backend**: NestJS
- **Language**: TypeScript
